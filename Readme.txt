This is a browser based program,
created in javascript, css and html.

You can generate quotes of wisdom by clicking the first button.

You can Add your name with the second.  Output is reset by clicking any other button or exiting.

You can calculate probability of an event occuring in the future using Beye's probabilities.  Wikipedia Beye's formula if you don't understand how this works.

Scroll down to see the output of each button.


To run the program:

Download or clone the repository.  Make sure you don't change the folder locations, file placement within the folders, or rename either.
If you do it won't work.

Next, simply run the fbuttonhtml.html with your favorite browser.
